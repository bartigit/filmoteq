# Filmoteka

A simple Android app in Kotlin using Jetpack Compose.  
Add, rate, and filter movies in your own collection.

---

## Features

- Add, edit, and delete movies
- Filter by category and status

---

## Technologies

Kotlin + Jetpack Compose
